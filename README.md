# Proyecto-Final

La Superfinanciera recibe miles de reclamaciones relacionadas con:

Tarjetas de crédito
Créditos de consumo
Créditos de vivienda
Canales digitales
Débito automático
Fraude

Objetivo General

Implementar una plataforma Big Data basada en Arquitectura Medallion para automatizar la ingesta, procesamiento, análisis y visualización de quejas financieras recibidas en la SFC y que se puedan visualizar las de Confiar Cooperativa Financiera

Actualmente los reportes son elaborados manualmente en Excel, generando:

Retrasos en la toma de decisiones
Falta de trazabilidad
Altos costos operativos
Dificultad para identificar patrones de quejas

Objetivos Específicos
Automatizar el cargue de datos.
Reducir tiempo de análisis.
Construir KPIs ejecutivos.
Implementar modelos predictivos.
Generar dashboards en tiempo real.
